MT evaluation

BLEU/NIST evaluation tools

Implements Papineni et al, 2002 BLEU (IBM), runs original NIST script mteval-v13a.pl.
On-line cgi packaging: University of Leeds, UK, Centre for Translation Studies, EU FP7 IAPP HyTra project

This implementation takes just one reference translation, should work with any language in utf8, etc.
